# Dashboard Sandbox

This directory is a lightweight Vite sandbox kept for UI experiments and isolated prototyping.

## Important

- It is not the main product UI.
- The production application lives in `../frontend`.
- CI and roadmap work should target `frontend/` unless a task explicitly calls out this sandbox.

## Local Use

```bash
npm install
npm run dev
```

## Recommendation

If a feature has graduated from experimentation, move it into `frontend/` and keep this directory small.
